Found a resource that would have helped you navigate life after matric? Add it here. This tool handles the GitHub stuff so you don't have to.

How it works

1
Get a GitHub PAT

Create a free GitHub account, then generate a Personal Access Token with repo scope. Instructions here →

Go to: Profile icon → Settings → Developer Settings → Personal access tokens → Tokens (classic) → Generate new token → tick repo → copy the token. Store it somewhere safe — you only see it once.

2
Load and edit the README

Click "Load README" to pull in the current content. Edit it using the rich text editor — no markdown knowledge needed.

3
Submit your contribution

Click "Commit & Create Pull Request". The tool forks the repo, commits your change, and opens a PR automatically. A maintainer reviews and merges it.

Project details

These are pre-filled — you don't need to change them.

⚠️ Your PAT is sensitive. It is used only for this session and is never stored anywhere.
