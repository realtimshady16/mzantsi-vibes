# TASKS

This file tracks what needs to be done on Mzantsi Vibes. Anyone is welcome to pick up a task — no experience needed for most of them.

**How to reference a task in your PR:** mention the task number in your pull request title or description, e.g. `Closes #003`.

**How to pick up a task:** leave a comment on the PR or reach out via [Instagram](https://www.instagram.com/mzantsivibes/) or [LinkedIn](https://www.linkedin.com/company/mzantsi-vibes/) so we know you're working on it.
